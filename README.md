# CatClock

A beautiful sky clock app builded with [flutter](https://flutter.dev/).

> This project is submitted to [Flutter Clock challenge](https://flutter.dev/clock)

## Getting Started

```bash
cd cat_clock
flutter create .
flutter run
```
